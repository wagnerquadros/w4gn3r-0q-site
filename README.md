# W4GN3R 0Q — Site Oficial

Página oficial do artista **W4GN3R 0Q**, desenvolvida para apresentar seus lançamentos, músicas, vídeos e redes sociais.

O projeto possui uma identidade visual inspirada nas cores e na estética do lançamento **Tocando Umazinha**.

## 🎵 Sobre o projeto

O site reúne informações sobre o trabalho musical de W4GN3R 0Q, incluindo:

- apresentação do lançamento;
- lista de músicas;
- acesso ao Spotify;
- links para YouTube e Instagram;
- layout responsivo para computadores e dispositivos móveis;
- interação para alteração da paleta de cores.
